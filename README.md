**Code for a greenhouse built on a ESP32-C6-DevKitC-1.**

<hr></hr>

![ESP32S3_Lite_Pinout.png](ESP32S3_Lite_Pinout.png)

### Default Connections

| GPIO Pin | Connection           |
|----------|----------------------|
| 19       | Top Light Relay      |
| 20       | Bottom Light Relay   |
| 21       | Fan Relay            |
| 47       | Pump Relay           |
| 12       | VRx (Joystick)       |
| 13       | VRy (Joystick)       |
| 14       | SW (Joystick)        |
| 1        | SDA (LCD & RTC)      |
| 2        | SCL (LCD & RTC)      |
| 4        | DHT-11               |
| 5        | Water Level (Gutter) |
| 6        | Water Level (Tank)   |
